1. 飞书应用初始化

一键安装与配置飞书 CLI

创建飞书应用前，请先在本地电脑准备飞书 CLI。请按下面任一方式完成。

方式一：手动安装

在本地终端中输入以下命令完成安装：

npx @larksuite/cli@latest install
方式二：Agent 自动安装

将以下引导语复制发送给你的 AI 工具，让它替你完成安装：

帮我安装飞书 CLI：https://open.feishu.cn/document/no_class/mcp-archive/feishu-cli-installation-guide.md

创建新应用

打开飞书创建链接后，请完成外部页面流程；回到本页后系统会自动检测结果。

用户授权

打开飞书授权链接后，请完成外部页面流程；原配置页会自动检测授权完成。

  • 创建公众号文章同步工作台和推送草稿表
  • 读取需要推送的文章内容
  • 写回推送成功或失败的结果
  • 创建自动推送和结果提醒流程
